# MealBuilder Full-Stack Application

A meal management system featuring a React frontend, Node.js/Express backend, and MySQL database.

## Features
- **Dashboard**: View summary stats and today's meal plan.
- **Recipe Library**: Add, view, and delete recipes.
- **Weekly Planner**: Schedule meals for specific dates and categories (Breakfast, Lunch, Dinner).

## Setup Instructions

### 1. Database Setup
- Open MySQL Workbench.
- Run the provided `1AT24CS138.sql` script to create the `meal_builder_db` and required tables.

### 2. Backend Setup
- Navigate to the root folder.
- Run `npm install`.
- Start the server with `node server.js`.

### 3. Frontend Setup
- Navigate to `meal-builder-react`.
- Run `npm install`.
- Start the app with `npm run dev`.
